Είχα γράψει σε προηγούμενο άρθρο για το πόσο πολύ «τιμωρείται» το ακίνητο και αντίστοιχα όποιος αντί να ξόδευε, έφτιαξε ή αγόρασε ένα σπίτι. Χθες είδα στις κυριακάτικες εφημερίδες τις λεπτομέρειες της νέας φορολογίας ακινήτων (ο νέος φόρος ονομάζεται ΕΦΙΑ -Ενιαίος Φόρος Ιδιοκτησίας Ακινήτων) μετά τις παρεμβάσεις «υπέρ αγροτών» και πήρα χαμπάρι τι γίνεται. Δεν χρειάζεται να είναι κανένας οικονομικός αναλυτής για να καταλήξει στο συμπέρασμα ότι πρόκειται ουσιαστικά για μια «μουλωχτή» λεηλασία που πάει να περάσει στο ντούκου και τα αποτελέσματά της θα τα δούμε σε 2-3 χρόνια. Εκτός κι αν πρόκειται για «σχέδιο μετακίνησης πληθυσμών» και αναζωπύρωσης της οικοδομικής δραστηριότητας στις ακριβές περιοχές.
Ας το δούμε όμως με ένα παράδειγμα που είδα σε κυριακάτικη εφημερίδα: Αν κάποιος έφτιαξε ή αγόρασε ή κληρονόμησε και κατοικεί σε ένα διαμέρισμα των 200 τ.μ. σε «ακριβή» περιοχή με αντικειμενική αξία π.χ. 5000&euro;/τ.μ. μόνο ο συμπληρωματικός φόρος που θα πληρώσει (επειδή έχει και καλά «μεγάλη ακίνητη περιουσία» δηλ. με τις τρέχουσες πλασματικές αντικειμενικές αξίες, η αντικειμενική αξία είναι πάνω από 300,000&euro;) θα είναι 3.900&euro; το χρόνο! Συν τον κανονικό (εκτός δηλ. από τον συμπληρωματικό) φόρο ακινήτων που θα είναι περίπου 3.000&euro;. Δεν ξέρω αν θα διατηρηθεί και το χαράτσι για να απογειωθεί ο σουρεαλισμός. Δηλαδή περίπου 7.000&euro; κάθε χρόνο που σημαίνει ότι εκτός από το όποιο δάνειο έχει και μια μηνιαία επιβάρυνση 600 &euro; το μήνα! Και αν έχει και τίποτε άλλο στην κατοχή του προστίθεται κι αυτό στον συμπληρωματικό φόρο. Και η πλάκα είναι ότι είχαν ακουστεί δηλώσεις ότι «θα επιμεριστούν τα βάρη σε περισσότερους για να ελαφρύνουν» και ήλπιζαν οι αστοί «προύχοντες» ότι θα μειωθούν τα χαράτσια για τα ακίνητά τους. Τελικά οι ίδιοι και οι ίδιοι, τα εύκολα θύματα θα την πληρώσουν (και δεν είναι λίγοι, υπολογίζονται σε 300.000 απ` ό,τι άκουσα στο ραδιόφωνο). 
Το κακό με τον αυξημένο φόρο ιδιοκτησίας είναι ότι τα λεφτά που ζητούνται δεν είναι επί κάποιου εισοδήματος που απορρέει από αυτό (π.χ. από εκμετάλλευση ενοικίων) και συνήθως δεν υπάρχουν. Πού θα βρίσκει επιπλέον 5-10 χιλ. &euro; κάθε χρόνο να δίνει ως φόρο ακινήτων ένας «προύχοντας» ελεύθερος επαγγελματίας που κυνηγάει τους πελάτες με το τουφέκι κι όταν βρίσκει τους κυνηγάει να τον πληρώσουν ή μισθωτός που βλέπει τον μισθό του να ψαλιδίζεται (και είναι τυχερός που έχει ακόμα δουλειά) ή συνταξιούχος ή άνεργος; Κι όταν κάποιος δεν αντέχει και πρέπει να πουλήσει το σπίτι του θα καταλάβει ότι η αντικειμενική αξία επί της οποίας μπαίνουν οι φόροι είναι πολλαπλάσια της κανονικής (σαν να φορολογείται κάποιος για διπλάσια ή και τριπλάσια εισοδήματα από αυτά που πραγματικά βγάζει). Και δεν θα βρει εύκολα κι αγοραστή και θα αρχίσει ο «νοικοκυραίος» να γίνεται φοροφυγάς. Υποθέτω μάλιστα ότι θα εμφανιστούν μετά από λίγα χρόνια εταιρίες που θα αγοράζουν τέτοια «φιλέτα» αντί πινακίου φακής (να κι η αναζωογόνηση της οικοδομής και η ανάπτυξη). Μέχρι τώρα βλέπαμε να «σκοτώνονται» καταναλωτικά αγαθά (π.χ. ακριβά αυτοκίνητα). Τώρα θα βλέπουμε να «σκοτώνονται» σπίτια, ακόμα και πρώτες κατοικίες στις κάπως ακριβές περιοχές (δεν ξέρω αν θα μπορούν να θεωρηθούν φτωχοί για να υπάρχει προστασία από πλειστηριασμούς πρώτης κατοικίας). Κι αν χάσεις τη δουλειά σου, πες ότι θα βρεις άλλη. Αν χάσεις το σπίτι σου; 
Άνθρωποι που έζησαν και μεγάλωσαν στο Ψυχικό, στη Φιλοθέη, στη Βούλα και κληρονόμησαν ή δανείστηκαν για να αποκτήσουν κάποιο ακίνητο ή ακίνητα θα είναι οι μελλοντικοί «πρόσφυγες του ΕΦΙΑ». Οι μονοκατοικίες με κήπο των περιοχών θα εξαφανιστούν καθότι δεν θα μπορούν να πληρωθούν οι φόροι τους. Γίνεται λοιπόν ουσιαστικά μια βίαιη προσαρμογή στα σύγχρονα οικονομικά δεδομένα που θα την πληρώσει αυτή τη φορά η μεσοαστική τάξη. Για πολύ πλούσιους κι εφοπλιστές δεν συζητάμε γιατί τα έχουν τα ακίνητα συνήθως γραμμένα σε offshores (εδώ που τα λέμε, αν δείτε στις περιοχές αυτές υπαίθριες διαφημίσεις με δικηγορικά γραφεία που ειδικεύονται στη σύσταση offshore μην απορήσετε: τα «κορόιδα» θα αρχίσουν να ξυπνάνε).
Για να βγουν λοιπόν τα νούμερα θα πρέπει κάποιοι να βγουν απ` τα σπίτια τους. Ακόμα και το κούρεμα των καταθέσεων φαντάζει πιο ορθολογικό καθότι εκεί τα λεφτά υπάρχουν, δεν είναι «εικονικά» και συνεισφέρουν όλοι ανάλογα με την πραγματική κι όχι εικονική τους δύναμη. Ούτως ή άλλως όμως κι αυτός ο νόμος κούρεμα των καταθέσεων είναι, καθότι όσοι καταφέρουν να ανταποκριθούν θα είναι εκείνοι που έβαλαν χέρι στις όποιες καταθέσεις είχαν κατορθώσει να μαζέψουν και διατηρούν ακόμα. Αν λοιπόν δείτε στις επόμενες εκλογές πρωτόγνωρα ποσοστά για τον ΣΥΡΙΖΑ σε Φιλοθέη, Κολωνάκι, Ψυχικό θα ξέρετε τον λόγο. Η Ελλάδα είναι άλλωστε μια πολύ ιδιαίτερη χώρα όπου ακόμα και ο κομμουνισμός με τον καπιταλισμό και τον σοσιαλισμό μπερδεύουν τα μπούτια τους.
Ελπίζω να μη βούτηξα πολύ το δάχτυλο στο βαζάκι του λαϊκισμού, αλλά να παρουσίασα τα πράγματα όσο αντικειμενικά επιτρέπουν οι ως τώρα πληροφορίες. Εν κατακλείδι, να φορολογείται η ακίνητη περιουσία αλλά θα πρέπει να γίνεται αυτό σε σωστή βάση κι όχι να «εξοντώνεται» όποιος δεν μπορεί να διαμαρτυρηθεί. Κάποιος θα πρέπει να ξαναδεί τους συντελεστές που έχουν μπει ή τουλάχιστον να εφαρμοστούν αυτοί με σωστές όμως αντικειμενικές αξίες που θα απεικονίζουν την πραγματική κατάσταση. Η φορολογία για να είναι δίκαιη πρέπει να είναι επί της πραγματικής περιουσιακής εικόνας κι όχι επί της εικονικής περιουσίας.
